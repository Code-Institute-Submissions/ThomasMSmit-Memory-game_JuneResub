Welcome to, memory game Assassin'S Creed edition

The goal of this game is to find all the matching images.

This will be a test for your memory.

The game starts with an equal number of cards, all the cards will be face down.

Start the game by clicking on the first 2 cards you think are a match ( You're a genius if you can get a match with the first 2 cards you click) , the selected cards will flip over revealing an image. If the second card selected does not match the first, then both cards will be flipped back face down.

Try to remember which image was in which spot to match them with their matching image when you find it.

This game requires good observation, a strong concentration and an amazing memory to win!

If you think you're ready for the challenge then press 'Start'!
